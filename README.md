# AtmoSphere (or your chosen project name)

An interactive, lightweight 3D Climate Globe built completely from scratch using vanilla HTML5 Canvas, CSS, and JavaScript. No external libraries (like Three.js) or dependencies required.

👉 [**Live Demo Link**](Your GitHub Pages URL here)

## 🌍 Features
* **Interactive 3D Globe:** Smooth mouse dragging, touch-tracking for mobile, and physics-based auto-rotation.
* **5 Key Climate Metrics:** Toggle and visualize data for CO₂ per capita, Renewable Energy %, Air Quality Index (AQI), Forest Cover, and Climate Risk.
* **Dynamic Country Analysis:** Hover to view tooltips with localized statistics and current net-zero targets.
* **Pure Performance:** Built using custom 3D coordinate mathematics projected onto a 2D canvas, ensuring fast load times and a tiny footprint.

## 📊 Data Sources
The dataset represents 18 major nations with data compiled from:
* Central Pollution Control Board (CPCB)
* International Energy Agency (IEA)
* World Bank & FAO
